Project deployment documentation
---------------------------------
Docker deployment
----------
1. rename .env.example to .env
2. docker-compose build
3. docker-compose up -d
4. go to the link http://localhost:5000/

Normal deployment
-----------------
1. npm i
2. rename .env.example to .env
3. npm start
4. go to the link http://localhost:5000/

I did full stack development here.
